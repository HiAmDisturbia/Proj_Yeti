Outil d'analyse du serveur yeti

Ce site web a été réalisé dans le cadre d'un projet à l'ENSG. Ce dernier consistait à réaliser un outil d'analyse de la clientèle du serveur yeti pour que les administrateurs puissent améliorer leur site.

Ce site permet de visualiser une carte de chaleur des zones les plus demandées par les utilisateurs ainsi que plusieurs staistiques à partir d'un fichier donné. La carte de chaleur permet d'observer les zones géographiques les plus demandées par les utilisateurs. Les staistiques permettents d'observer les types d'erreur les plus répendu.
